# üéÆ Trapped ‚Äì A C++/SFML Narrative Quiz-based Game

**Trapped** is a narrative-driven quiz and escape-room-style game designed by a team of nine Computer Science & IT students from UKZN. The project explores interactive storytelling, player choice, and immersive mechanics as the player navigates mental challenges to escape a fictional trap. The game uses multiple storyline paths, puzzles, and inventory interactions to engage and test the player.

## üß© Game Overview
Players wake up in a dark basement, facing a series of narrative challenges that must be solved to escape. Each decision leads to different outcomes, guided by the player's choices and puzzle-solving ability. The storyline and gameplay options are drawn dynamically from external **text files**, allowing modular and expandable script paths.

## üíª Technologies Used
- **Language:** C++  
- **Library:** SFML (Simple and Fast Multimedia Library)
- **Accessibility Tool**: Microsoft Azure Text-to-Speech API (fully implemented for voice narration and enhanced user experience)  
- **Development Tools:** Visual Studio 2022
